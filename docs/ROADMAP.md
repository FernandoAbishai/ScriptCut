# Roadmap

ScriptCut is a local-first AI post-production tool for creators who want to edit spoken video as text, generate polished clips, and export social-ready videos without uploading raw media.

## Phase 1: Core Editing Feels Real

- Edited preview that skips deleted ranges
- Autosave and recovery
- Export progress, logs, cancellation, and retry
- Reliable waveform and timeline behavior

## Phase 2: AI Suggestions Are Editable

- Filler-word review queue
- Editable clip drafts
- AI-generated titles, hooks, captions, descriptions, and hashtags

## Phase 3: Social-Ready Export

- Vertical and square presets
- Reframe controls and safe-area preview
- Caption styling
- Background removal and replacement
- Batch clip export

## Phase 4: Trustworthy Open Source

- One-command setup and environment diagnostics
- Clear provider and privacy settings
- Deterministic `.scriptcut` project format
- CI and smoke checks
- Contributor-friendly issues and documentation
