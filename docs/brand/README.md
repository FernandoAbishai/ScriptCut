# ScriptCut Brand Assets

The primary ScriptCut mark is **Transcript Slice**. It represents the
product's defining interaction: editing a video by editing its transcript.

## Primary Assets

- `frontend/public/brand/scriptcut-mark.svg`: compact application mark.
- `frontend/public/brand/scriptcut-wordmark.svg`: mark plus ScriptCut wordmark.
- `build/icon.svg`: source used to generate the macOS application icon.

The SC Edit Loop and Timeline Cut concepts are retained here as exploratory
concepts. They are not product marks and should not replace the primary icon
without an intentional brand decision.

## Usage

Use the compact mark in application chrome, favicons, and square app icons.
Use the wordmark on the welcome screen, README, release pages, and other
public-facing surfaces where the product name needs to be visible.
